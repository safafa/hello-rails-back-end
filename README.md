## Getting Started

To get a local copy up and run the app, follow these simple example steps.

### Prerequisites

    Ruby 
    Rails 
    Postgres 

### Setup

In your terminal clone the repo.
To install all dependencies, run:

```
  bundle install
```

Create database and Messages table
```
  rails db:create
  rails db:migrate
  rails db:seed
```
To start the server run:

```
  rails s -p 3001
```
